# Asuka Player 对外展示版

## 产品简介
Asuka Player 是一款专注本地视频播放体验的 Android 播放器。  
我们希望把“打开就能看、操作直觉、播放稳定”做到极致，让本地视频管理和播放变得更轻松。

## 核心价值
- 轻快：常用操作路径短，重点功能一目了然
- 好用：手势与控制逻辑清晰，学习成本低
- 稳定：播放链路清晰，状态可恢复，长期可维护
- 可持续：模块化架构，便于持续迭代和优化

## 主要功能亮点
- 本地媒体库浏览  
  按文件夹组织视频内容，快速定位与进入播放。

- 沉浸式播放体验  
  针对全面屏与手势导航场景进行界面适配，减少视觉干扰。

- 手势驱动的高频操作  
  支持快进、亮度、音量、缩放、平移、双击和长按等手势控制。

- 可配置的播放器行为  
  提供续播、默认播放速度、自动画中画、后台播放等常用设置能力。

- 信息反馈直观  
  控制层、进度反馈、状态提示保持简洁且可感知，降低误操作成本。

## 体验设计方向
- 一致性：视觉语言与交互规则统一，减少认知切换
- 相关性：图标与文案高度对应，功能意图清晰
- 克制感：避免过度动效和噪声，优先保证操作效率
- 可达性：核心操作触达简单，单手场景友好

## 技术与工程能力（对外摘要）
- 采用 Kotlin + Jetpack Compose 构建现代化 UI
- 基于 Media3 构建播放能力，兼顾稳定性和扩展性
- 多模块分层设计，并对启动编排、会话规划、状态恢复做了职责拆分
- 配套测试与质量清单，持续优化可靠性与性能

## 适用场景
- 日常本地视频播放
- 视频素材快速预览
- 长视频连续观看与续播
- 需要高频手势控制的移动端观看场景

## 项目原则
- 以用户体验为中心，优先解决真实使用问题
- 以工程质量为底线，避免短期方案累积技术负担
- 坚持合规开发，不复制受限代码与资源

## 未来规划（方向）
- 进一步优化冷启动与首屏进入流畅度
- 持续提升手势与控制层联动体验
- 完善后台播放与系统级媒体控制能力
- 强化媒体管理效率与个性化配置能力
